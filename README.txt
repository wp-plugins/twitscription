=== Twitscription ==

Contributors: natambu
Tags: user, admin, access

A WordPress plugin that allows users to login user Twitter OAUTH authentication.

== Installation == 

This section describes how to install the plugin and get it working.

1. Download the archive and expand it.
2. Upload the *twitscription* folder into your *wp-content/plugins/* directory
3. Goto http://twitter.com/oauth_clients and create an application named "Users Blog"
4. In your *WordPress Administration Area*, go to the *Plugins* page and click *Activate* for *Twitscription*
5. In your *WordPress Administration Area*, go to the *Settings* section and click *Twitscription*
6. Fill in the consumer key and secret
7. Check enabled and update settings
